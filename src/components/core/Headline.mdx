The Headline component creates `<h1>`, `<h2>`, etc. tags based on the level prop set. Use the size prop to change the size to a different level depending on your needs.

```js live
<div>
    <Headline level={1} size={1}>
        H1 Level 1 Size 1
    </Headline>
    <Headline level={2} size={2}>
        H2 Level 2 Size 2
    </Headline>
    <Headline level={3} size={3}>
        H3 Level 3 Size 3
    </Headline>
    <Headline level={4} size={4}>
        H4 Level 4 Size 4
    </Headline>
    <Headline level={5} size={5}>
        H5 Level 5 Size 5
    </Headline>
    <Headline level={6} size={6}>
        H6 Level 6 Size 6
    </Headline>
</div>
```
